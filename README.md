# Rapido Earnings Dashboard

An Excel-based dashboard designed to track, analyze, and summarize daily ride earnings from Rapido.

## Project Overview

This project was created to practice building a complete Excel reporting workflow, starting from raw ride data and ending with summarized insights through pivot tables and dashboards.

## Features

* Ride-wise earnings tracking
* Daily earnings summary
* Pivot table analysis
* Dashboard for quick performance review
* Structured data entry and reporting

## Skills Demonstrated

* Microsoft Excel
* Pivot Tables
* Data Organization
* Dashboard Design
* Reporting and Analysis
* Formula-Based Calculations

## Workbook Structure

| Sheet         | Purpose                          |
| ------------- | -------------------------------- |
| Ride_Entries  | Stores individual ride records   |
| Daily_Summary | Summarizes daily earnings        |
| Pivots        | Pivot table analysis             |
| Dashboard     | Visual overview of earnings data |

## Purpose

The objective of this project was to gain hands-on experience in transforming raw operational data into meaningful summaries and dashboard-based insights using Microsoft Excel.
